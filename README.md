# SpikeStorm

**SpikeStorm** é um jogo simples criado com **Pygame**, inspirado em *Geometry Dash*.  
O objetivo é pular sobre os obstáculos (spikes) e sobreviver o máximo possível conforme a velocidade aumenta.  
É uma versão **bem simplificada**, ideal para aprendizado, modificação e diversão casual.

---

## Características principais
- Estilo **arcade/plataforma**, inspirado em *Geometry Dash*
- Jogabilidade rápida e minimalista
- Aumenta de dificuldade com o tempo
- Sistema de pontuação baseado na distância percorrida
- Sons, música e gráficos leves e retrô

Mecânicas de jogo
O personagem corre automaticamente para a direita.

Pule para evitar os spikes.

Cada obstáculo ultrapassado aumenta a pontuação.

Colidir com um spike resulta em Game Over.

A velocidade aumenta conforme o jogador sobrevive mais tempo.
